# Kuroi Dokutsu

A small dungeon-crawling game for the **Neo Geo Pocket Color**, inspired by *Cave Noire*.

> ⚠️ **Work in progress.** This is a game currently in development — not a finished or
> released title. Features are incomplete, some assets are placeholders, and things
> may change or break between commits. It's shared openly so others can follow along
> and learn from the code.

## About

*Kuroi Dokutsu* ("black cave") is a homebrew game built from scratch for the Neo Geo
Pocket Color, using the public hardware specification. You explore procedurally
generated caves, one screen at a time — collecting, avoiding, and finding your way
back out.

The project is still being built, so expect rough edges. It's here as a work in
progress, not a polished release.

## About the name

*Cave Noire* — the game that inspired this one — is a Japanese title with a French
name. So, as a small joke, I flipped it around: a game made in France with a
Japanese name. *Kuroi Dōkutsu* (黒い洞窟) simply means "black cave."
